# Getting Started
### Checking out this branch 
After you have run `git clone`, you will need to checkout this branch with <br>
`git checkout gameplay-2` <br>
if you are on `git 2.0+`, you may need to do <br>
`git switch -c gameplay-2`
### To run a local deployment of this branch:<br>

`cd gameplay-2` <br>
`npm i` <br>
`npm run dev` <br>




# Dependencies
`npm 10.1.0+`
